# Skill Genome — Tech Stack (Backend)

## 1. Backend Framework
**Language:** Python  
**Framework:** FastAPI  

- High-performance async APIs  
- Native AI/ML compatibility  
- Auto API documentation  

---

## 2. Core Responsibilities

- Resume parsing & extraction  
- GitHub activity ingestion  
- AI questionnaire processing  
- Skill signal computation  
- Alignment engine  
- Corporate Readiness Index calculation  
- Confidence modeling  
- Data storage  

---

## 3. AI / ML Layer

| Component | Technology |
|-----------|------------|
| Embeddings | Sentence Transformers |
| Skill Inference | Custom ML pipelines |
| Alignment Logic | Vector + rule-based |
| Confidence Modeling | Probabilistic scoring |
| Trend Analysis | Time-series methods |

---

## 4. Data Ingestion

| Source | Method |
|--------|-------|
| GitHub | GitHub REST API v3 |
| Resume | PyMuPDF / pdfplumber |
| Questionnaire | Internal APIs |

---

## 5. Processing & Analysis

- Python AST parsing  
- Git commit analysis  
- Skill graph (NetworkX)  
- Temporal tracking  

---

## 6. Database Layer

| Type | Tech |
|------|-----|
| Primary | PostgreSQL |
| Graph | Neo4j (optional) |
| Cache | Redis |

---

## 7. API Layer

- FastAPI REST endpoints  
- JWT authentication  
- Rate limiting middleware  
- Celery + Redis for async tasks  

---

## 8. CRI & Signal Engine

- Aggregates signals  
- Computes CRI  
- Produces explanations  
- Adds confidence scores

## 9. Security
